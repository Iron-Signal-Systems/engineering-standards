# Changelog

## 1.0.0 — Initial standard

- Establishes ISRAS v1.
- Defines native-first and VM-capable reproducibility without universal
  container dependence.
- Defines portable, canonical, specialized, historical, fresh-clone, release,
  deployment, recovery, and operational validation layers.
- Defines repository governance, secure development, supply-chain assurance,
  migration integrity, historical integrity, performance governance, Windows AD
  lab integration, and acceptance evidence.
- Adds adoption, policy, environment, fresh-clone, checkpoint, migration
  manifest, and acceptance evidence tooling.
