# Acceptance Records

This directory retains sanitized, durable acceptance plans, historical
records, evidence identities, and correction records.

Large or restricted logs remain in approved evidence storage or CI
artifacts. Durable records identify those artifacts by SHA-256 and source
boundary.

For v1.0.1 and later, the verified signed annotated release tag is the
authoritative acceptance-decision object. An in-tree acceptance plan defines
the criteria before candidate freeze but does not claim the final decision.
