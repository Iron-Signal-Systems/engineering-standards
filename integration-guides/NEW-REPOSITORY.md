# New Repository Integration

1. Create the empty project repository.
2. Clone it and create `dev`.
3. Add the project skeleton.
4. Run `tools/isras/adopt.py` from `engineering-standards`.
5. Customize environment profiles and portable checks.
6. Commit source, tests, fixtures, validation, and documentation together.
7. Run portable validation from each approved development system.
8. Run fresh-clone validation.
9. Add workflows in observation mode.
10. Enable rulesets after stable runs.
11. Formally accept repository assurance before the first release claim.
