## Required focus

- Go formatting, vetting, tests, race tests, and dependency verification;
- deterministic inventory and document generation;
- Draw.io and generated-document source retention;
- schema validation;
- stable identifiers;
- per-file SHA-256 manifests;
- byte-identical regeneration where the format permits it;
- explicit normalization where formats contain timestamps or unstable metadata;
- historical phase gates;
- clean-clone portability.

Adopt in observation mode before requiring checks.
